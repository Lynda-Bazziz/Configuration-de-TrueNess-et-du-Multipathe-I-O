# Configuration-de-TrueNess-et-du-Multipathe-I-O
Ce projet décrit la mise en place d’une infrastructure de stockage réseau utilisant TrueNAS avec le protocole iSCSI, ainsi que la configuration du Multipath I/O côté client.

L’objectif est de permettre l’accès à des volumes de stockage distants comme s’ils étaient des disques locaux sur une machine Windows Server.
